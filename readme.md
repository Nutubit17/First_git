git 사용법
---
* git은 뭔가 파일을 만들고 저장했다면 git add로 그 파일을 저장해줘야 함. **//git** **add** **"파일명"**
* 추가가 완료되면 그 상태에서 commit 필요함. **//git** **commit** **-m** **"메세지"**
* 커밋이 끝나면 remote에 올리는 작업을 해야 함. **//git** **push** **origin** **master**
